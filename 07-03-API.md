# Tiny Hippo 当日推荐

## 获取对应日期推荐列表 [GET] (/recommendation/{?date})

**参数**

- `date (Date)`，当日推荐的日期

**属性**

- `image_url (string)`，推荐图像 URL 地址 
- `title (string)`，推荐语标题
- `subtitle (string)`，推荐语副标题
- `description (string)`，推荐描述
- `food (food)`，推荐食物，`(food)` 类型，包含一下属性
  - `description (string)`，食物描述
  - `image_url (string)`，食物描述图像 URL 地址 
  - `name (string)`: 食物名称 
  - `number (int)`: 食物数量, 
  - `price (int)`: 食物价格

**Response 200**

  - Header

    ```
    Last-Modified: 2018-04-17
    ```

  - Body

    ```json
    {
      "title": "recommendation title e.g. 初夏美食",
      "subtitle": "recommendation subtitle e.g. 春夏交际的清凉食物",
      "description": "recommendation descrption e.g. 夏天到了 blahblah",
      "image_url": "xxx.png",
      "food": {
        "description": "description 1", 
        "image_url": "xxx.jpg", 
        "name": "food name", 
        "number": 0, 
        "price": 24
      }
    }
    ```

# Tiny Hippo 菜谱

**参数**

- `class (string, optinal)`，菜品类别

- `foods (list)`，应类别的食物列表，列表成员为食物对象 `food 类型`，包含以下属性
  - `description (string)`，食物描述
  - `image_url (string)`，食物描述图像 URL 地址 
  - `name (string)`: 食物名称 
  - `number (int)`: 食物数量, 
  - `price (int)`: 食物价格


## 获取菜品的分类列表 [GET] (/menu/)


**属性**

- `url`，类别对应的子类 url
- `name (string)`，类别中文名称


**Request (application/json)**

  - Header

    ```json
    Accept: application/json
    ```

  - Body

    ```json
    {}
    ```

**Response 200**

  - Header
  
    ```
    Accept: application/json
    ```
  
  - Body
  
    ```json
    {
      "class_list": [
        {
          "name": "class name e.g. 早餐",
          "url": "class url e.g. breakfast"
        }
      ]
    }
    ```

# Tiny Hippo 订单详情 

## 获取订单 id [POST] (/order/)

**属性**

- id (int)，随机生成的订单 id

**Request (application/json)**

  - Header

    ```
    Accept: application/json
    ```

  - Body

    ```json
    {}
    ```

**Response 200**

  - Header
  
    ```
    Last-Modified: 2018-04-17
    ```
  
  - Body
  
    ```json
    {
      "id": 123456
    }
    ```

## 获取订单详情 [GET] (/order/{?id})

参数

- id (int, optional)，订单编号

属性

- foods (list)，订单食物列表，包含 (food) 类型的对象
  - `description (string)`，食物描述
  - `image_url (string)`，食物描述图像 URL 地址 
  - `name (string)`: 食物名称 
  - `number (int)`: 食物数量, 
  - `price (int)`: 食物价格
- total_price (int)，总价格
- create_time (date)，订单生成时间
- modified_time (date)，订单上次编辑时间

**Request (application/json)**

  - Header

    ```
    Accept: application/json
    ```

  - Body

    ```json
    {}
    ```


**Response 200**

  - Body
  
    ```json
    "foods": [
        {
          "description": "description 1", 
          "image_url": "xxx.jpg", 
          "name": "food name", 
          "number": 1, 
          "price": 24
        }
      ],
      "total_price": 120,
      "create_time": 2017-12-12-12:02:00,
      "modified_time": 2017-12-12-14:08:00
    }
