# Tiny Hippo 菜谱 (/menu/{?class})

参数

- `class (string)`，菜品类别

属性

- `name (string)`，类别中文名称
- `foods (list)`，应类别的食物列表，列表成员为食物对象，包含以下属性
  - `description (string)`，食物描述
  - `image_url (string)`，食物描述图像 URL 地址 
  - `name (string)`: 食物名称 
  - `number (int)`: 食物数量, 
  - `price (int)`: 食物价格
