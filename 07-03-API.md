  - Body
  
    ```json
    {
      "message": "",
    ```

# Tiny Hippo 当日推荐 (/recommendation/{?date})

参数

- `date (Date)`，当日推荐的日期

属性
- `image_url (string)`，推荐图像 URL 地址 
- `title (string)`，推荐语标题
- `subtitle (string)`，推荐语副标题
- `description (string)`，推荐描述
- `food (food)`，推荐食物，`(food)` 类型，包含一下属性
  - `description (string)`，食物描述
  - `image_url (string)`，食物描述图像 URL 地址 
  - `name (string)`: 食物名称 
  - `number (int)`: 食物数量, 
  - `price (int)`: 食物价格

## 获取推荐列表 [GET]

  - Header
  
    ```json
    Last-Modified: 2018-04-17
    ```
  
  - Body
  
    ```json
    {
      "title": "recommendation title e.g. 初夏美食",
      "subtitle": "recommendation subtitle e.g. 春夏交际的清凉食物",
      "description": "recommendation descrption e.g. 夏天到了 blahblah",
      "image_url": "xxx.png",
      "food": {
        "description": "description 1", 
        "image_url": "xxx.jpg", 
        "name": "food name", 
        "number": 0, 
        "price": 24
      }
    }
    ```

# Tiny Hippo 菜谱 (/menu/{?class})

参数

- `class (string)`，菜品类别

属性

- `name (string)`，类别中文名称
- `foods (list)`，应类别的食物列表，列表成员为食物对象 `food 类型`，包含以下属性
  - `description (string)`，食物描述
  - `image_url (string)`，食物描述图像 URL 地址 
  - `name (string)`: 食物名称 
  - `number (int)`: 食物数量, 
  - `price (int)`: 食物价格


## 获取对应菜品分类列表 [GET]

- Request (application/json)
  - Header
  
    ```json
    Accept: application/json
    ```
  
  - Body
  
    ```json
    {}
    ```

- Response 200
  - Header
  
    ```json
    Last-Modified: 2018-04-17
    ```
  
  - Body
  
    ```json
    "foods": [
        {
          "description": "description 1", 
          "image_url": "xxx.jpg", 
          "name": "food name", 
          "number": 0, 
          "price": 24
        },
        {
          "description": "description 2", 
          "image_url": "xxx.jpg", 
          "name": "food name", 
          "number": 0, 
          "price": 24
        },
      ], 
      "name": "classification name"
    }
    ```
